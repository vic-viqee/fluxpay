# FluxPay — web

This folder contains a minimal Next.js application with Tailwind CSS configured.

Quick start

1. Change to the `web` directory:

```bash
cd web
```

2. Install dependencies:

```bash
npm install
# or
# pnpm install
# or
# yarn install
```

3. Run the dev server:

```bash
npm run dev
```

The app uses the `app/` router. Open http://localhost:3000 to view it.

Notes

- `next`, `react`, `react-dom`, `tailwindcss`, `postcss`, `autoprefixer`, and `typescript` are set to `latest` in `package.json` so `npm install` will fetch current versions.
- If you prefer JavaScript-only, remove `typescript` and delete `tsconfig.json` / `.d.ts` files.
